## Overview

<a href="#">coconutBattery</a> is a Mac utility developed by coconut-flavour that provides detailed battery health diagnostics for Mac computers and connected iOS and iPadOS devices, revealing the technical condition data that Apple exposes partially in System Information but never presents in an organized, historically tracked, easily readable format. Used by Mac owners who want to understand their battery's true condition, IT administrators managing device fleets, and anyone considering purchasing a used Mac or iPhone, coconutBattery provides the objective battery health picture that informed hardware decisions require.

The <a href="#">battery design capacity versus current capacity</a> comparison is the most fundamental metric coconutBattery provides. Every lithium battery is manufactured with a specific milliamp-hour design capacity — the amount of charge it holds when new. As charge cycles accumulate, the battery's maximum charge capacity degrades below that design specification. coconutBattery displays both values alongside the percentage of original capacity remaining, expressing battery health as a concrete measurement rather than the coarse categorical indicators Apple's own tools provide. <a href="#">Cycle count</a> displays the exact number of complete charge cycles the battery has accumulated — the primary technical factor in battery lifespan — alongside Apple's rated maximum cycle count for the device model, giving precise context for how much of the battery's rated service life has been consumed.

<a href="#">Temperature monitoring</a> displays the current battery temperature in real time, flagging elevated temperatures that indicate the battery is working harder than normal — a condition associated with heavy processing loads, hot ambient environments, and battery degradation. Sustained high battery temperatures accelerate capacity loss, making temperature awareness valuable for users who want to preserve battery health over the device's lifespan. <a href="#">Charging state detail</a> shows whether the battery is currently charging, at what wattage, and what the current charge level is in milliamp-hours in addition to the standard percentage — providing the granular charging status that the macOS battery indicator deliberately simplifies.

<a href="#">iOS and iPadOS device monitoring</a> extends coconutBattery's diagnostics to any iPhone, iPad, or iPod touch connected to the Mac via USB. The same design capacity, current capacity, cycle count, and temperature data available for the Mac's internal battery reads from connected iOS devices, making coconutBattery a comprehensive battery health station for the entire Apple device ecosystem a user owns. <a href="#">Device comparison</a> displays data for the Mac and multiple connected iOS devices simultaneously, providing a unified battery health overview across all devices in a single interface.

## Key Features

- <a href="#">Design vs current capacity comparison</a> shows original milliamp-hour design capacity against current maximum charge capacity with health percentage
- <a href="#">Exact cycle count display</a> reveals the precise number of accumulated charge cycles alongside the device model's Apple-rated maximum cycle specification
- <a href="#">Real-time temperature monitoring</a> tracks battery temperature with alerts for elevated conditions that accelerate long-term capacity degradation
- <a href="#">Granular charging wattage display</a> shows current charge rate in watts and milliamp-hours alongside standard percentage for detailed charging status
- <a href="#">iOS and iPadOS device diagnostics</a> reads identical battery health metrics from any iPhone iPad or iPod touch connected via USB to the Mac
- <a href="#">Multi-device simultaneous display</a> shows Mac and all connected iOS device battery data in a single unified interface for fleet overview
- <a href="#">Menu bar status indicator</a> displays current battery health percentage and charge status in the menu bar for instant always-visible monitoring
- <a href="#">Battery health history tracking</a> records capacity and cycle count measurements over time for visual degradation trend analysis
- <a href="#">Pre-purchase device inspection</a> reveals used Mac and iPhone battery condition objectively for informed second-hand hardware buying decisions

## Additional Information

coconutBattery's value extends beyond curiosity about battery health into practical decision-making territory. A Mac showing 78% battery capacity with 800 cycles has quantifiably less remaining service life than one showing 92% capacity with 200 cycles — and coconutBattery provides exactly these numbers in a format that makes the comparison objective rather than speculative. For anyone buying a used Mac or iPhone, the two minutes spent running coconutBattery can reveal battery replacement needs that significantly affect the device's real-world value and near-term cost of ownership.

The application's continued usefulness over years of ownership reflects the ongoing relevance of battery health data — a metric that becomes more important as a device ages and approaches the point where battery replacement or device upgrade decisions need to be made on solid technical evidence rather than subjective feel.
